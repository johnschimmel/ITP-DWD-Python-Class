Description

If you ever had a need to collect information from users on the web or use external data in your project, understanding web development will make your life much easier. The class will explore interaction between server-side and client-side of web development using Python and JavaScript. On the client-side, we will cover traditional JavaScript and the jQuery library to manipulate browser content, create and trigger page events and make AJAX data requests. Developing with Python on the server-side, we will explore receiving input from a user then querying and saving that data to a database, and finally, returning the appropriate content to the client, i.e. HTML or JSON. The websites we use today are rarely on a single database, we will focus on consuming data APIs from websites like Foursquare (for location information), Facebook (for social graph). Going further, we will create custom data APIs for use at ITP and open to the public.

Students must have prior HTML & CSS experience or be willing to learn it outside of class. Introduction to Computational Media or prior object oriented programming knowledge is required.

  • Respect your fellow students' questions and comments.
  • Be 'here'. Participate in class.
  • Assignments are given for each class to practice and try out recent topics. All assignments are required and must be completed.

  • No computer usage in class that is not relevant to the current topic.
Grading

Syllabus

September 03 Week 1: Introduction to web development

Introduction to web servers. Semester overview. Getting environments set up.

September 10 Week 2: Python Crash Course

Python overview, variables, conditions, lists, dictionaries, loop and tabs.

September 17 Week 3: WWW! Introduction to Flask, Git and Heroku

Getting our first web server set up and running. "Hello World" wide web literally.

September 24 Week 4: Flask - Request, Responses & Templates

Setting up Flask for templates (html) and static files (images, css and js). Separating code, data and design.

October 01 Week 5: Databases - Introduction to MongoDB

Setting up MongoDB with Heroku Addons. Thinking about data structure & schemas.

October 08 Week 6: More Databases - Inserting & Form Validation

Form validation when accepting user data

October 22 Week 7: More Databases - Reference fields and Forms that love you back

Queries and Reference fields with Mongo

October 29 Week 8: Introduction to JSON, API's & REST

JSON, your new best friend. Sharing and using remote data easily with API's and REST. Creating a simple API to share your data.

November 05 Week 9: Using Remote API's & more JSON

How to use remote web apis. Examples with Foursquare and Twilio.

November 12 Week 10: Users, Sessions & Authentication

Libraries and functions to add user registration and authentication for managing access to content.

November 19 Week 11: Uploading Files and Images / Amazon S3

Uploading files and images from a browser to Heroku and saving on Amazon S3. Saving and querying images by user from database.

November 26 Week 12: Realtime Web - Instant Gratification

Using realtime communication between from server to client, client to client or server to server.

December 03 Week 13: Processing and Arduino as Web Servers and Clients

Getting Processing and Arduino to provide and consume web API's. Spacebrew is a great tool to make the interaction between different applications and devices happen and persist.

December 10 Week 14: return {'TheEnd': True}

Present final projects and say our final thoughts
