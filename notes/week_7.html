Ordering results from Query

Ordering w/ MongoEngine User Guide

.order_by allows you to sort a field or fields in ascending/descending order.

# descending timestamp, most recent first
ideas = models.Idea.objects().order_by('-timestamp')

# alphabetical by title  ( + sign is optional )
ideas = models.Idea.objects().order_by('title')

# multiple fields
ideas = models.Idea.objects().order_by('title','-timestamp')

Filter Your Data

How to query your data with MongoEngine - User Guide

Basic

# get all
ideas = models.Idea.objects()

# get Tony's ideas
ideas = models.Idea.objects(creator='Tony')

# get Tony's most recent idea
idea = models.Idea.objects(creator='Tony').order_by('-timestamp').first()

# or in another way
idea = models.Idea.objects.get(creator='Tony').order_by('-timestamp')

# multiple field query - Tony's physical computing ideas
ideas = models.Idea.objects(creator='Tony',categories='physical computing')

Search text

You can search the text of your documents with the __contains filter.

# case sensitive
ideas = models.Idea.objects(title__contains='Sneaker')

# case insensitive
ideas = models.Idea.objects(title__icontains='sneaker')

Search inside embedded document

Search inside an embedded document for text

# Ideas that Tony has commented on
ideas = models.Idea.objects(comments__name='Tony').order_by('-comments__timestamp')

Example from user guide

class BlogPost(Document):
    title = StringField()
    date = DateTimeField()

    @queryset_manager
    def objects(doc_cls, queryset):
        # This may actually also be done by defining a default ordering for
        # the document, but this illustrates the use of manager methods
        return queryset.order_by('-date')

Reference Fields in Models

Your documents can be related / associated with other documents using the Reference Field. You define reference fields in your models as a field type. Great documentation in the user guide.

# Models
class User(Document):
    name = StringField()

class Page(Document):
    content = StringField()
    author = ReferenceField(User)

# create user and page
tony = User(name="Tony Smith")
tony.save()

post = Page(content="Test Page")
post.author = tony #  tony is the author
post.save()
