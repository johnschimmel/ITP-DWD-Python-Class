Overview

You may have need for a user to submit a file or image via a form to your web application, this is called a file upload. Web servers can accept form submissions with binary data (the contents of a file).

A server set up to receive files from a form submission might do the following

  • Accept files via POST.
  • Verify the MIME/file type is accepted.
  • Read the contents of the file.
  • Save the file to a specific directory. Or upload to a storage service like Amazon's S3.
  • Record the filename, final filepath or other information to a database.

Prepping your models.py

Sample of Image model (models.py)

class Image(mongoengine.Document):

    # we store only the filename, not the actual file
    filename = mongoengine.StringField()

    title = mongoengine.StringField(max_length=120, required=True)
    description = mongoengine.StringField()
    postedby = mongoengine.StringField(max_length=120, required=True, verbose_name="Your name")
    tags = mongoengine.ListField( mongoengine.StringField())

    # Timestamp will record the date and time idea was created.
    timestamp = mongoengine.DateTimeField(default=datetime.now())

Setting up your form

Two things will be changed in your normal form

1) Form encoding, enctype="multipart/form-data" added as attribute of the main <form> tag.

<form method="POST" enctype="multipart/form-data" role="form">

2) The file upload input box, type='file'.

HTML

<div class="form-group">
    <label for="fileupload">Upload an image file</label>
    <input class="btn btn-info" id="fileupload" name="fileupload" type="file">
</div>

When using WTForms/Jinja

<div class="form-group">
    {{ form.fileupload.label }}
    {{ form.fileupload(class="btn btn-info") }}
</div>

Displaying Photos

1) Query the database for Images you need to display

2) Inside the template render the filename appended to the S3 bucket path

{% for img in images %}
    <a title="{{img.title}}" alt="Title:{{img.title}}. Description: {{img.description}}" href="https://s3.amazonaws.com/dwd_uploads/{{img.filename}}">
    <img src="https://s3.amazonaws.com/dwd_uploads/{{img.filename}}" width="480"></a>
    <br>
    <h3>{{img.title}}</h3>
    <p>{{img.description}}
        <br>
        {{ img.postedby }} - {{ img.timestamp.strftime("%m/%d/%y")}}
    </p>

{% endfor %}
